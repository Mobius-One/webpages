Download for Firefox at the add-on store: here

Download from source: here


Install the extension, open SR and click on the extension icon.
On desktop the user agent will be spoofed so mobile mode will enabled.
It will also perform small changes to page so it avoids the "rotate your device" warning and make some elements of the game display properly.

Where does it work?
NOT AVAILABLE YET, WAITING REVIEW Chrome: It works on desktop. On mobile devices such as tablets, it works with mobile chromium based browsers (e.g: Kiwi Browser) that allow extensions, unfortunately the official version does not.
Firefox: It works on desktop (specially the Twitch login that doesn't normally work when mobile mode is enabled). On mobile devices such as tablets it works with the mobile Firefox Beta that allows the installation of add-ons.
