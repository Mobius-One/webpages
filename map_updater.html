Map Node Updater

Follow the instructions below to update the map nodes

  • Click "Download formatted JSON"
  • Wait a few seconds while the file is downloaded and processed
  • Replace the old file with the new file at /extension_folder/icons/loyalty_chests.json
  • The extension can't manipulate local user files outside the Chrome storage, so you have to do it manually.
  • Even tho the map nodes file belongs to the extension, it is still outside the Chrome storage.
