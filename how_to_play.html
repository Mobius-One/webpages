SRHelper is a click emulator for the game Stream Raiders.

Download for Google Chrome at the extension store here


To get started, load the game after installing the extension and click on the extension icon on the extensions tab. That will open the AutoRaiders popup screen.
The popup has several settings to tailor the gameplay to your needs. Some of them are mandatory such as which units are allowed to be placed.
Below is the explanation of all the extension settings and how they work:

Switch idle captain: The extension works with both live and offline captains. It will always switch to a campaign captain Switch idle captain after 15 minutes, the idle states are: if a captain doesn't find a battle, doesn't start a battle or doesn't collect the reward.
Leave slot empty if there aren't any loyalty captains to switch After removing an idle captain, by default the extension will seek a captain with loyalty: diamond, gold, silver, then bronze. If it doesn't find any it will seek a favorited captain. If doesn't find any it will randomly select one, if this option is enabled it won't select a random captain if a loyalty/favorite captain isn't available.
Buy Scrolls Automatically buy scrolls from the store.
Buy extra $100 scrolls Refresh the store for 100 coins and buy the scrolls.
Collect quests Collect completed quests.
Collect daily reward Collect free daily event rewards such as cupcakes.
Collect battlepass Collect completed battlepasses (it will not collect potions if the amount of potions is maxed out).
Preserve diamond loyalty Skip loyalty chest battles if the captain doesn't have diamond loyalty (skin chest, token chest, scroll chest, boss chest, super boss chest).
Placeable units - keep in mind that these settings don't apply to dungeons as the cooldown for dungeons does not apply to the other modes
Use common units Allow the placement of common units.
Use uncommon units Allow the placement of uncommon units.
Use rare units Allow the placement of rare units.
Use legendary units Allow the placement of legendary units.
Throttle for special game modes: By default only one unit is placed on special mode, those can be overwritten with the following options.
Unlimited duels Place more than one unit on duel mode.
Unlimited clashes Place more than one unit on clash mode.
Unlimited dungeons Place more than one unit on dungeon mode.
Epic units/Potion usage
No potions No potions will be used.
Potions at 45 If there's a minimum of 45 potions, epic units will used.
Potions at 100 If there's exactly 100 potions, epic units will be used.
Color code: From time to time you'll see the slots changing colors, here's what they mean. Units won't be placed on colored slots, new placement attempts are made every 5 minutes
Red It means that a slot is running a special mode (dungeons/PVP) while a second captain switched to the same mode, the second captain will be colored red and no units will be placed to avoid resetting keys for example. It also means that a captain was running a special mode and switched to campaign, some users dislike this because it locks the slot for 30 minutes, no units will be placed to preserve the slot.
Purple There aren't any units available to be placed on the available markers, this can be due to units on cooldown or unmatching markers.
Gold If the preserve diamond loyalty option is enabled and the captain is on a loyalty chest and doesn't have loyalty no units will be placed.
Wipe captain button Reset the red, purple and gold states as well as the pause and offline buttons.
▶ Play/Pause ⏸ button on the slot Pause individual slots so they can be manually managed or prevent joining the next battle (keep in mind that the idle switcher will overwrite this setting, disable it as well to prevent that).
ENABLED/DISABLED idle switcher buttons Prevent offline switching of individual slots when a captain is idling so they can be manually managed.
